## 快速更新文件至 gitHub

	1. `cd ~/local/` 将 sc 文件放在里面，并且 `pwd` 。

	4. `gedit ~/.bashrc` 添加 `export PATH="把这段文字换成local的绝对路径哈哈:$PATH"`

	3. `source ~/.bashrc`

	2. `cd` 到你 gitHub 的地址，并且 `pwd` 复制路径。

	3. 打开 sc 文件，将第4步复制的路径替换。

	4. 如果遇到权限不足，则 `chmod +x ~/local/sc`

	4. 你可以在任何路径上`sc "这里写你做了什么"`
